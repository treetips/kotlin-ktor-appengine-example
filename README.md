# ktor on google appengine example

## Features

* Java v8
* kotlin v1.3.11
* Gradle v5.0
* ktor v1
* Google appengine standard environment for java8
* jetty9

## Requirement

* macOS
* IntelliJ IDEA

## Usage

```bash
# run local application
./start.sh
open http://localhost:8080/json

# deploy appengine
./deploy.sh
```
